# learn_words_flutter

本项目是使用 Flutter 开发的一个背单词 App。

以词书为维度可对单词进行导入、背诵等操作；


本项目的灵感来源：[YouZack 背单词](https://bdc2.youzack.com/Recitation/Home)，部分设计及数据参考或来源于该网站，注意：本项目目前非 YouZack 背单词官方版本。
